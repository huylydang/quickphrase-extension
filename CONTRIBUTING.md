# Contributing to QuickPhrase

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Clone your fork:
   ```bash
   git clone git@github.com:YOUR_USERNAME/quickphrase-extension.git
   ```
3. Load in Chrome as unpacked extension (see [README](README.md))
4. Create a branch: `git checkout -b feature/my-feature`

## Development

- **No build step** — edit files directly, reload extension in Chrome
- **Test changes** by selecting text on any page and using the context menu
- After making changes, reload the extension on `chrome://extensions/`

## Pull Requests

1. Keep changes focused — one feature/fix per PR
2. Test on at least 2-3 different websites
3. Update README if you add new features
4. Describe what you changed and why

## Code Style

- Plain JavaScript (no TypeScript, no build tools)
- Use descriptive variable names
- Comment non-obvious logic
- Keep functions small and focused

## Reporting Bugs

Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Chrome version
- Screenshot if applicable

## Feature Requests

Open an issue with the `feature` label. Describe:
- The problem you're solving
- Your proposed solution
- Why it's valuable for other users
